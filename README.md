# mvts-edge-simulator
Desarrollador Edge &amp; Middleware (El "Dueño" de los Datos) para un MVTS

Herramientas: Python, Docker, RabbitMQ.

Misión: Esta persona será la encargada de levantar el contenedor de RabbitMQ (con Docker) y de programar el script en Python que simula los camiones.

Hito de éxito: Lograr que la terminal muestre que los JSON se están publicando correctamente en la cola de RabbitMQ.

Ventaja: Es el rol ideal para quien prefiera trabajar rápido en la consola y no lidiar con bases de datos ni diseños visuales.
